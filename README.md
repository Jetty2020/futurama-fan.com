## 1. 목표와 기능

- simple apis의 [futurama api](https://sampleapis.com/api-list/futurama)를 이용하여 futurama 팬페이지 구현
  - 페이지네이션을 이용한 에피소드 
  - 동적라우팅을 이용한 캐릭터 상세페이지

<br>

- Deploy
  - Vercel

### 2 프로젝트 구조

```
.
├── README.md
├── next-env.d.ts
├── next.config.js
├── node_modules
├── package.json
├── public        
├── src
│   ├── components    //   추상화 요소
│   ├── constants     //   상수 지정
│   ├── hooks         //   커스텀 훅스
│   ├── pages         //   페이지 
│   ├── styles        //   기본 스타일
│   ├── types         //   타입 관심사 분리
│   └── utils         //   유틸 함수
├── tsconfig.json
└── yarn.lock
```
